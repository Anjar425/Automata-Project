# Automata-Project

## How To Run

1. Clone This Repository

```bash
  git clone https://github.com/Anjar425/Automata-Project.git

```
2. Change Directory
```bash
  cd Automata-Project
```
3. Install Requirements 
```bash
    pip install -r requirements.txt
```
4. Run Program 
```bash
    flask run
```
5. Then Open Link that Show in Terminal. In Example 
```bash
    http://127.0.0.1:5000/
```
